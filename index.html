Contact of network in primary school





Link to dataset

This dataset is part of a study of contact networks in a primary school, as reported in the paper High-Resolution Measurements of Face-to-Face Contact Patterns in a Primary School. The dataset includes information about the contacts of students with each other as well with teachers. The students are grouped into classes. Each class is shown in different color. The purpose of this kind of visualization is to report properties of the contact patterns between school children that are relevant for modeling the propagation of diseases and for evaluating control measures.

The average degree centrality is the average of number of links incident upon all nodes. The value is 49.992 for this dataset. For this network it gives the number of contacts that a student has with all other students. Since mostly the nodes are connected to their own class strongly, most of the degree contribution comes from their own class.


Network Diameter gives us Betweenness Centrality which measures how often a node appears on the shortest paths between nodes in the network. By coloring the graph using betweenness centrality as node ranking we see that the inner nodes have larger values than the outer nodes which is intuitive since inner nodes would appear more often between shortest paths of different nodes. For this network it signifies which students interact more with other class students.


Eigen Vector centrality is a measure of node importance in a network based on a node's connections. A high eigenvector score means that a node is connected to many nodes who themselves have high scores. For this network we can see that the inner nodes have more eigenvector centrality value then the outer nodes.This can be seen in the graph below. It means the students that interact with their own class students as well as other class students have higher values of eigenvector centrality which means their measure of influence is higher.



Description

Eigen vector values range from 0.1 to 1. These values are represented using size. Greater the node size higher the value for that node. From the above legend we can see that node size of 30 corresponds to eigenvector centrality of 1 and node size of 15 corresponds to eigenvector centrality of 0.1. Another channel is also used to represent this - color brightness. Brighter the green color higher the value and vice versa. The variance of color is shown in the legend above.
